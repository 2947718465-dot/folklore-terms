# 中国民俗学术语库

**Folklore Terminology Database**

由史骞升编撰维护。收录 5,769 条中国民俗学及相关学科核心术语，涵盖民俗事象、理论术语、学科概念、研究方法、研究关键词、学者与学术体制六大类，三层分类体系，中英双语对照。

## 技术说明

- GitHub Pages 部署
